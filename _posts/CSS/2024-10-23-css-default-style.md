## Default styles

> Refer -> [Browser Default Styles](https://browserdefaultstyles.com/)

CSS resets are stylesheets [containing CSS rules] that alter or remove the default stylesheet set by browser [this is referred as user-agents rules].

They're just used to address style inconsistencies.
